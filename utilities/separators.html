Separators are used for separating items in a linear list with a character. The chararctor may be a •, |, or —. Using this utility will allow for more space between those characters than an HTML space would typically allow.

sba- u- separator--[VALUE]
Namespace
Prefix
BEM syntax

The value is a number of 1 thru 7, with that number being a multiple of 8.
