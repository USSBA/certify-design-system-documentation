Certify Styleguide

Welcome to the certify.SBA.gov styleguide. Here you will find guidance on components used in the certify.SBV.gov application. These components are built with the U.S. Web Design Standards.

The U.S. Web Design Standards are an excellent resource for design guidance and packaged components. The standards are meant to be built upon.

Task Panels

Task panels are interactive menus that contain bulk actions for items they relate to. The panels are revealed by clicking the three-dot icon. In order for them to function properly, the toggle button must have an aria-controls value that corresponds to the id of a task-panel-content.

Example

8(a) Initial Application
  • Begin processing this case
  • Add a note
  • Request info from firm
  • Send message to firm
  • Return to firm for revisions
  • Close this case

Code


  <div class="task-panel">
    <button type="button" class="usa-button-unstyled task-panel-toggle" aria-expanded="false" aria-controls="task-panel1">actions</button>
    <div id="task-panel1" class="task-panel-content">
      <ul class="task-panel-menu">
        <li class="task-panel-menu__item">
          <a href="#" class="task-panel-menu__link"><strong>Begin processing this case</strong></a>
        </li>
        <li class="task-panel-menu__item">
          <a href="#" class="task-panel-menu__link">Add a note</a>
        </li>
        <li class="task-panel-menu__item">
          <a href="#" class="task-panel-menu__link">Request info from firm</a>
        </li>
        <li class="task-panel-menu__item">
          <a href="#" class="task-panel-menu__link">Send message to firm
        </li>
        <li class="task-panel-menu__item">
          <a href="#" class="task-panel-menu__link">Return to firm for revisions</a>
        </li>
      </ul>
      <hr>
      <ul class="task-panel-menu">
        <li class="task-panel-menu__item">
          <a href="#" class="task-panel-menu__link--emergency">Close this case</a>
        </li>
      </ul>
    </div>
  </div>

Required Compents

  • Toggle
  • Panel
  • Menu

Optional Components

  • Emergency actions

    Emergency actions are red links that involve actions such as closing or deleting.

  • Horizontal rules

    If a menu has more than 3 items, and one of those items is an "emergency action," then the two menus should be used seperated by a horizontal rule.

Variations

There are variations on the toggle button available that can be applied using class names on the toggle button.

Vertical dots

The dots can be rotated vertically to save horizontal space, such as when the task panel is used on cards. This is done by adding a class name of task-panel-toggle--vertical to the toggle

<button type="button" class="usa-button-unstyled task-panel-toggle task-panel-toggle--vertical" aria-expanded="false" aria-controls="task-panel3">actions</button>

Smaller buttons

Buttons can be made smaller to conserve space, such as when the task panel is used in cards and tables. This is done by adding a class name of task-panel-toggle--small to the toggle

<button type="button" class="usa-button-unstyled task-panel-toggle task-panel-toggle--small" aria-expanded="false" aria-controls="task-panel3">actions</button>
